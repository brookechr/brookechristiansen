Work on Climate Braintrust Slack Bot

This project concerned a Slack bot that allows users of the Work on Climate Slack workspace of 4,000+ members to search terms and have their synonyms come up in the results, rather than just exact string matching. I've been contributing to this project from December 2020 to February 2021.

I worked on the project voluntarily with a group of industry developers and an industry product manager. As the only student in the group, it was been a valuable learning experience to work with such experienced and helpful people.

We used Python, pytest, Airtable, Google App Engine, and GitHub for this project. I set up the AirTable for the definitions and synonyms, which are crowd-sourced. A lot of the other work that I did was in the form of pair programming over video calls.
