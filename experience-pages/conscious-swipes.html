Conscious Swipes App

This was a project my group and I worked on for Georgia Tech's HackGT. It is a prototype of an Android app that lets users swipe (Tinder-style) on people with similar values they'd like to discuss. If two users swipe right on each other (meaning they are both indicating they would like to chat with each other), then they will be 'matched' and the second person will receive the first's Zoom link, where they can video chat and discuss whatever they would like to. We saw this as a way for users to make connections and have meaningful discussions safely despite the pandemic.

Given more time to complete this project, we would implement an algorithm for matching users based on how they rank each of their values, embedded video chat, login and sign-up screens, and a settings page.

We used Flutter, Dart, the flutter_tindercard widget, Figma, VSCode and GitHub to complete this.
